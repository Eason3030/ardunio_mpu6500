自動升降輔助輪系統（Arduino）

本專案為一套整合 OLED介面、MPU6500姿態偵測、紅外線遙控、霍爾感測、繼電器控制、線性制動器 的自動化輔助輪系統。整體邏輯以安全性、可視化呈現與穩定運作為核心。

⸻

🚲 專案功能總覽

1. OLED 全介面顯示系統
	•	左上角顯示 MPU6500 的即時角度
	•	顯示輔助輪狀態：上升 / 下降 / 準備中
	•	顯示霍爾感測器狀態（例如煞車偵測、位置觸發）
	•	顯示繼電器動作（每次咖一聲即更新畫面）
	•	客製化 UI：
	•	開機動畫
	•	跑馬條 loading
	•	系統檢查
	•	狀態切換動畫

⸻

🔧 系統邏輯與流程

2. MPU6500 姿態偵測邏輯
	•	讀取角度（Pitch / Roll）
	•	顯示於 OLED 左上角
	•	角度可成為判斷輔助輪是否可安全上升的參考（依需求擴充）

⸻

3. 紅外線遙控控制輔助輪升降
	•	IR 遙控器按鍵 → Arduino → 控制繼電器
	•	觸發線性制動器上升或下降
	•	OLED 會同步顯示系統狀態（UP/DOWN）

⸻

4. 繼電器控制邏輯
	•	Arduino 接收 IR 或霍爾訊號後觸發
	•	控制線性制動器電源方向
	•	每次動作都會產生一聲“咖”，OLED 即時更新狀態

⸻

5. 霍爾感測器安全檢測
	•	可做為「緊急煞車」觸發
	•	或判斷輪子是否完全上升/下降到指定位置
	•	偵測後即時更新 OLED、停止線性制動器並鎖定狀態

⸻

🧪 其他硬體測試項目
	•	使用外接 Arduino 測試線性制動器 PWM，目標找出最適行程速度
	•	已完成與機械科合作，確定 M8 螺帽與制動器機構的實體結合方式
